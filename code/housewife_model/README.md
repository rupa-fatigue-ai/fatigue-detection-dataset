# Housewife Fatigue Detection Model

## Description
This project implements a fatigue detection model using physiological signals (such as ECG and GSR) collected from household subjects. The model applies machine learning techniques to classify fatigue levels in daily-life environments.

## Dataset
The dataset used in this study is available in this repository:
datasets/housewife_dataset/housewives_dataset.zip

It contains physiological signals with corresponding fatigue labels.

## Methodology
- Data preprocessing and normalization  
- Feature extraction from physiological signals  
- Machine learning-based classification  

## Requirements
Install dependencies using:

pip install -r requirements.txt

Common libraries include:
- numpy  
- pandas  
- scikit-learn  
- matplotlib  
- seaborn  

## How to Run
1. Open the notebook:
   housewife_fatigue.ipynb  

   OR  

2. Run the script:
   python housewife_fatigue.py  

## Notes
This model is designed for fatigue monitoring in household environments and can be extended to similar real-world scenarios.
