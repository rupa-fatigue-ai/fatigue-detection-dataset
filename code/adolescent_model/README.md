# Adolescent Fatigue Detection Model

## Description
This project implements a fatigue detection model for adolescents using physiological and/or behavioral data. The approach leverages machine learning techniques to classify fatigue levels.

## Dataset
The dataset used in this study is not publicly shared due to privacy constraints. However, it consists of physiological and behavioral features collected from adolescent subjects.

## Methodology
- Data preprocessing and normalization
- Feature extraction
- Machine learning-based classification

## Requirements
Common Python libraries required:
- numpy
- pandas
- scikit-learn
- matplotlib
- seaborn
  
## How to Run
1. Open the notebook:adolescent_fatigue.ipynb
OR run: python adolescent_fatigue.py
