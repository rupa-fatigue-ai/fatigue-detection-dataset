# Worker Dataset

## Description

This dataset contains multimodal physiological signals collected from 25 subjects. It includes electrocardiogram (ECG), electroencephalogram (EEG), and galvanic skin response (GSR) signals.

## Data Composition

* Subjects: 25
* Signals:

  * ECG
  * EEG
  * GSR
* Labels:

  * Questionnaire-based annotations

## File Structure

* Each CSV file corresponds to one subject.
* Each file contains synchronized physiological signals.

## Purpose

This dataset is intended for fatigue detection and analysis using multimodal physiological data.

## Usage

The dataset can be used for:

* Feature extraction
* Machine learning model development
* Comparative analysis of physiological signals

## Notes

* All subject identifiers are anonymized.
* Data is provided in raw signal format.
