# Physiological Dataset

## Description

This dataset contains physiological signals collected from multiple subjects for fatigue detection analysis. It includes electrocardiogram (ECG) and galvanic skin response (GSR) signals.

## Data Composition

* Signals:

  * ECG
  * GSR
* Labels:

  * Condition-based annotations

## File Structure

* Each CSV file corresponds to one subject.
* Files include timestamped physiological signal recordings.

## Purpose

This dataset is designed for fatigue detection using physiological signals.

## Usage

The dataset can be used for:

* Signal processing
* Feature extraction
* Machine learning applications

## Notes

* Subject identities are anonymized.
* Data is provided in structured CSV format.
