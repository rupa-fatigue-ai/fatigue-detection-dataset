# fatigue-detection-dataset
# Multimodal Physiological Datasets for Fatigue Detection

This repository contains datasets used for fatigue detection research based on physiological signals.

## Repository Structure

* `datasets/worker_dataset/`
  Multimodal dataset containing ECG, EEG, and GSR signals with labels.

* `datasets/housewife_dataset/`
  Physiological dataset containing ECG and GSR signals with labels.

## Usage

Each dataset is organized independently and can be used for training and evaluating machine learning models for fatigue detection.

For detailed information, refer to the README file inside each dataset folder.
